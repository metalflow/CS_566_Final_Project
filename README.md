# CS_566_Final_Project

A speech to text webpage that should take in voice input and attempt to locate Movie Theaters near the user. Should also provide basic information about particular movies.
Initial code stolen from https://github.com/caterinasworld/web-speech-demo
Altered to meet "Hey Siskel" project needs by adding the Compromise and Synonym NPM packages
and then making use of those packages to do limited Natural Language Proecssing to performe the following basic functions:

1. Provide the duration of a partcular movie
2. Provide release date of a particular movie
3. Provide a Movie Review for a particular movie
4. Locate a particular Movie Theater in a Particular City
